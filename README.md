# openAPI_python

python 스크립트로 openAPI를 활용해 데이터베이스에 카테고리와 도서 정보를 insert합니다.

### how to use

1. python 가상 환경 구성
    
    ```bash
    python -m venv myenv
    
    source myenv/bin/activate
    
    pip install -r requirements.txt
    ```
    
2. secret.py 정보 확인
3. category.py 스크립트 실행 
4. book.py 스크립트 실행
